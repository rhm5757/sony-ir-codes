Sony Blu-ray (26.226; 26.234; 26.242; 26.151; 26.135; 26.164)

 

This includes the usual 3 Blu-ray modes, plus some other device codes presumably using the same set of commands:

The UHP-H1 Blu-ray player (Remote: RMT-VB210*), which for some reason got its own device code and model number prefix used on nothing else.

The portable Blu-ray players (Model: BDP-SX*, Remote: RMT-B113*).

The Home Entertainment Server (Model: HES-V1000, Remote: RMT-HS001A), which combines a Blu-ray megachanger and server with large hard disk.

 

The original source wasn’t exactly clear, but presumably, mode 3 (26.242) is the default for the recorders, similar to how it works with DVD.  Apparently there was a recorder with a hard disk, probably only in Japan, but the mode switch commands are in other Sony remotes (usually as F1 and F2).

 

Some of these commands are attempts at translating the Japanese names for the functions, and if anyone knows more about what some of these buttons do, e-mail me.

 

Thanks to Scott Harris for the BDP-S1 command codes and Guy Lambert for the recorder information.
