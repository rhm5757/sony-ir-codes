Sony Vaio PC (26.178; 26.51; 26.59)

 

The older Sony Vaio PCs that come with remotes have only 1 mode, using device 26.178.  That became mode 1 on newer remotes, while 26.51 is mode 2 and 26.59 is mode 3.  The current Sony PC remotes, however, aren’t Sony at all.  They use the RC6-6-32 protocol, which is typically used by Phillips and the brands it owns.  Since these really aren’t Sony remotes and I know little about the RC6-6-32 protocol, I have chosen not to include it on the website.  I have not gotten any requests for information about Sony PC remotes so it probably won’t be a problem.
