Parameters

Parameter Name

Description and Format

<ASCII>

An ASCII character.  Usually followed by x# in the parameter listing to indicate the maximum string length.

<audio in status>

Always 1?

<input>

<audio input>

<video input>

The input’s code number.  Here is the list:

0 = Tuner

1 = Phono

2 = CD

3 = DAT

4 = MD/DAT

5 = Tape

16 = Video 1

17 = Video 2

18 = Video 3

19 = Video 4

21 = DVD/LD

22 = TV/Sat

25 = DVD/LD (alt.)

254 = nothing (used for audio inputs with no video)

<input device>

The device code number of the device the receiver thinks is attached on the other end of the current input.  It’s not very clear how it determines this, especially for non-Sony devices.  255=nothing.
