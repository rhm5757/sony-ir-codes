Sony CD (17, 26.153, 26.202; 57, 26.161; 81, 26.169)

 

Sony has three CD codes.  Most devices use 17.  All three use the same commands, just with different device codes as is the case with most Sony stuff.  The codes are also similar to the DVD codes, not surprisingly.  Device 17 is CD1, device 57 is CD2, device 81 is CD3.  These codes are all from OFA’s advanced code list for cd/0000 except for a few.  CD-R devices have a switch for CD/CD-R, but the command codes are different between modes for the same button.

 

I’m pretty sure that the one time I found a CD/MD combo remote using the CD-R codes, it was the wrong remote for the device.  I think all CD/MD combos use the CD codes in CD mode and the MD codes in MD mode like you would expect.

 

I got a tip on a ccf that contained a mode 2 device code for 26.153, not surprisingly 26.161.  I have confirmed that the mode 3 device code is 26.169.

 

Olive = obsolete and replaced by other codes

Orange = text entry secondary names – These vary greatly between players and many recent players have them assigned to unusual codes.  I am not going to attempt to list every primary name/text entry name combination.  When looking for the command code for a button, look for the primary name (in gray) in the table.
