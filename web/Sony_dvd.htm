Sony DVD (26.73, 26.98; 26.34, 26.3; 26.250, 26.11; 26.18)

 

This is pretty much every Sony DVD button ever, as well as some undocumented commands I found.  This comes from my own experience as well as numerous .ccf files.  I discovered that Sony has a separate code for portable DVD players (called DVD2 on the AV3000) that use 26.18 as the device code in place of 26.73.  I don’t know if there is a portable DVD equivalent of 26.98.

 

I found some codes in the discrete code section of Remote Central that are listed as “mode 3” and use a new device code, 26.250.  I assume that it is the DVD3 code, so I have added the device code to this page.

 

The new Sony DVD recorder is out, and I have learned the new codes from its remote.  I now have the correct DVD mode 2 code, 26.34, as well as the secondary device code for mode 2 and 3.  By default it uses the mode 3 codes, 26.250 and 26.11.

 

DVD camcorders use the mode 2 codes (26.34 and 26.3).

 

Green = camcorder only
