Sony Portable DVD (26.18)

 

The codes for portable DVD players use their own device code (26.18) that has slightly different command codes from the normal DVD command codes.  The “Advance” and “Options” keys were notably moved from the secondary device code in DVD to the only device code in Portable DVD, so presumably there is no secondary device code.
